License Guide

ArmorPilot Enterprise

How ArmorPilot licensing works

ArmorPilot uses a request-and-activate model. You generate an activation request tied to your installation ID, send it to the vendor, and receive a license key in return. The key is then pasted into the console to unlock enterprise features.

1

Request a License

Fill in your organization details and send the request

a

Go to Access Control → License

In the top navigation, click Access Control, then in the sub-navigation click License.

b

Click "Download Activation Request"

This opens the Request a License form. Fill in your organization name, contact person, email, edition (Professional / Enterprise), duration, and number of nodes.

Required fields: Organization name, Contact name
Optional: Tax ID, Phone, Country, Address, Notes
c

Send via Email or Download

At the bottom of the form you have two options:

✉
Send via Email — opens your mail client with the form pre-filled. Attach the downloaded varmor-activation-request.json before sending.
⬇
Download Request — saves varmor-activation-request.json to your computer, then closes the modal. Attach this file to your email manually.
d

Send email to vendor

Send the email (with the JSON file attached) to tuancr2001@gmail.com. The vendor will verify the request and reply with a license key.

Typical response time: 1–2 business days
2

Activate a License

Paste the key you received from the vendor

a

Check your email for the license key

The vendor will reply with a license key in the body of the email. It starts with ARMORPILOT1. followed by two base64 segments separated by dots.

ARMORPILOT1.eyJjdXN0b21lcil6Ii4uLiJ9.signature…
b

Go to Access Control → License

Click Access Control in the header, then License in the sub-navigation.

c

Paste the key into "Install License"

In the Install License card on the right side, paste the full license key (the entire ARMORPILOT1.xxx.yyy string) into the text box.

Paste the complete key — do not trim any characters. Legacy signed JSON format is also accepted if provided by the vendor.
d

Click "Save License"

Click the Save License button. The License Status card will refresh and display your plan details, expiry date, node limit, and enabled features.

License Status card shows edition, expiry, node & policy limits
Header badge updates to your edition (e.g. Enterprise)
All enterprise features are immediately available

Troubleshooting

"Invalid license" error after Save

Make sure the key was not wrapped to a new line by your email client. Copy the entire key as a single line with no spaces.

"License does not match installation" error

The license is bound to your Installation ID. If you re-installed or migrated the server, you need to request a new license with the new Installation ID.

Email client did not open after "Send via Email"

Your browser may have blocked the mailto: link. Use "Download Request" instead and email the JSON file manually to tuancr2001@gmail.com.

License expired

Request a renewal license using the same flow. Select "Renewal" as the request type in the form. Your existing Installation ID stays the same.

Approaching node or policy limit

Check the License Status card for current usage. Remove unused policies or request an upgrade to increase limits.

Where is the Installation ID?

Go to Access Control → License. Click "Copy Installation ID". It is automatically included in the activation request JSON file.
